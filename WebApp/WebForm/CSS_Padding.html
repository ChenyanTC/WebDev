CSS Padding

CSS padding Properties

The CSS padding properties are used to generate space around content.

The padding properties set the size of the white space between the element content and the element border.

The setting of padding properties is similar to margin.

This element has a padding of 50px.

Padding values

  • length - specifies a padding in px,pt,cn,etc.
  • % - specifies a padding in % of the width of the containing element.
  • inhert - specifies that the padding should be inherited from the parent element.
