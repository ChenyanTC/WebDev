Border Style

The border-style property specifies what kind of border to display.

The following values are allowed:

  • solid 实线
  • double 双实线
  • groove 3D凹槽 The effect depends on the border-color value
  • ridge 3D菱形边框 The effect depends on the border-color value
  • inset 3D凹边 The effect depends on the border-color value
  • outset 3D凸边 The effect depends on the border-color value

Border Color

The broder-color property is used to set the color of the four borders.

The color can be set by:

  • name- like "red"
  • Hex- like "#ff0000"
  • RGB- like "rgb(255,0,0)"
  • transparent

The border-color property have from one to four values(for the top border,right border,bottom border,and the left border).

If border-color is not set,it inherits the color of the element.

Border Width

The border-width property specifies the width of the four borders.
