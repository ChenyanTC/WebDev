CSS Margins

CSS Margin Properties

The CSS margin properties are used to generate space around element.

The margin properties set the size of the white space OUTSIDDE the border.

Margin values

Al the margin properites can have the following values.

  • auto - the browser calculates the margin.
  • length - specifies a margin in px,pt,cm,etc.
  • % - specifies a margin in % of the width of the containing element.
  • inhert - specifies that the margin should be inherited form the parent element.
