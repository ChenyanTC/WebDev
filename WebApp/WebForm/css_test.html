CSS Selector

The element selector.The element selector selects elements based on the element name.

The id selector.To select an element with a specific id, write a hash(#) character,followed by the id of the element.

The class selector.The class selector selects element with a specific class attribute.To select element with a specific class,write a period(.) character,followed by the name of the class.

The HTML elements can also refer to more than one class.

The class can follow the element,like p.center.(The center is the class name.)

The Grouping Selectors can group the selectors,to minimize the code.To group selectors,separate each selector with a comma.

CSS How to

There are three ways of inserting a style sheet.

  • External style sheet
  • Internal style sheet
  • Inline style

Multiple Style Sheet

The value from the last read style sheet will be used.
  • If the internal style is defined after the link to the external style sheet,the internal style will be effective.
  • If the external style sheet is defined after the internal style sheet,the external style will be effective.
  • The Inline style has the highest priority.

The Cascading Order

Order:

  1. The inline style.
  2. The internal and external style sheets(in head section)
  3. The browser default.

CSS Colors

Colors in CSS are most often specified by:

  • A valid color name - like "red"
  • an RGB value - like "rgb(255,0,0)"
  • a HEX value - like "#ff0000"

CSS Background

The CSS background properties are used to define the background effects for elements.

CSS background properties:

background-color
background-color proerties specifices the background color of an element.
background-image
background-image property specifies an image to use as the background of an element.
background-repeat
specifies the image repeated way(repeat-x;repeat-y;no-repeat)
background-attachment
To specifies that the background image should be fixed(will not scroll with the rest of the page)
background-position
The position of the image is specifies by the background-position property.
