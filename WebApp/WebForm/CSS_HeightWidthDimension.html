CSS Height and Width Dimensions



CSS Dimension Proerties

The CSS dimension properties allow you to control the height and width of an element.

This eleemnt has a width of 100%

Setting height and width

The height and width properties are used to set the height and width of an element.

The height and width can be set to auto (this is default.Means the brower calculates the height and width), or be specified in length values,like px,cm,etc. or percent(%) of the containing block.

This div element has a height of 100 pixels and a width of 500 pixels.

Setting max-width

The max-width property is used to set the maximum width of an element. If the browser's windows smaller than the max-width,the width of the element will reduce to align with the browser's window.
